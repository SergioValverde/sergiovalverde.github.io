# Sergio Valverde — Cybersecurity Portfolio

Personal cybersecurity website and technical blog focused on Application Security, Cloud Security, DevSecOps, detection engineering, and hands-on security research.

Live site:

```text
https://sergiovalverde.github.io/
```

## Purpose

This site is used to publish practical security write-ups, cloud security labs, defensive architecture notes, and technical research based on hands-on experimentation.

The objective is to document real learning, practical implementation, and security reasoning across offensive and defensive topics.

## Main Topics

* AWS security architecture
* DevSecOps and security automation
* Application Security
* Web application security
* GitHub and software supply-chain security
* Detection engineering
* Windows security
* Cloud-native defensive controls
* Security lab documentation

## Content Philosophy

The content is intended to be practical and evidence-based. Posts should explain:

1. The security problem being addressed.
2. The architecture or lab environment used.
3. The implementation steps.
4. The detection, prevention, or validation logic.
5. The limitations, risks, and defensive takeaways.

## Intended Audience

This site is primarily intended for:

* Security engineers
* Cloud security practitioners
* DevSecOps engineers
* Detection engineers
* Application security professionals
* Recruiters and hiring managers reviewing practical security work
